Designed and implemented a full-stack, tech-domain search engine with an automated web crawler that continuously collects gadget and technology content. Built a robust data pipeline using SQLite for storage and a high-performance C++ core engine implementing inverted indexing, TF-IDF ranking, category filtering, and optimized query processing. Incorporated a secure land-chain inspired data flow for reliability and traceability of crawled data updates. Developed an intuitive Flutter frontend providing fast, responsive search experiences with real-time results and filtering capabilities.
